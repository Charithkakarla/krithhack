# VNR Smart School Dashboard (Frontend)

This frontend is a Vite + React dashboard UI for the VNR Smart School admin panel.

## Tech Stack

- React 18
- Vite 5
- Lucide React (icons)
- Recharts (line and bar charts)

## Run Locally

```bash
npm install
npm run dev
```

Open the URL printed by Vite (usually http://localhost:5173).

## Production Build

```bash
npm run build
npm run preview
```

## Implemented UI Sections

- Dark navy sidebar with all required navigation icons and support card
- Header panel with date, notification bell, avatar, and student view selector
- Five KPI cards
- Attendance and marks charts (last 7 days)
- Top performing students table
- Students needing attention list with progress bars
- Classwise attendance bar chart
- Upcoming events panel
- Quick actions and recent activities cards
